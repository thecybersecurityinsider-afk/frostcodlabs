📋 FILE CONNECTIONS CHECKLIST:

✓ index.html imports styles.css
  Line: <link rel="stylesheet" href="styles.css">
  
✓ index.html imports script.js
  Line: <script src="script.js"></script>
  
✓ All three files must be in the SAME FOLDER
  
✓ File names are CASE-SENSITIVE on Mac/Linux:
  • Must be: styles.css (not Styles.css or STYLES.CSS)
  • Must be: script.js (not Script.js or SCRIPT.JS)
  • Must be: index.html (not Index.html or INDEX.HTML)

💡 TROUBLESHOOTING:

Problem: CSS not loading (looks unstyled)
Solution: Make sure styles.css is in the same folder as index.html

Problem: JavaScript not working (menu doesn't toggle)
Solution: Make sure script.js is in the same folder as index.html

Problem: Names incorrect
Solution: Verify exact file names (lowercase, no spaces)

Problem: Still not working
Solution: 
1. Close browser completely
2. Clear browser cache (Ctrl+Shift+Delete)
3. Open index.html again
4. Try a different browser (Chrome, Firefox, Edge)
