🐛 TROUBLESHOOTING:

Problem: Website looks plain/no colors
─────────────────────────────────────
Solution: 
  1. Make sure styles.css is in the same folder as index.html
  2. Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R)
  3. Clear browser cache
  4. Try a different browser

Problem: Menu doesn't toggle on mobile
──────────────────────────────────────
Solution:
  1. Make sure script.js is in the same folder as index.html
  2. Check browser console for errors (F12)
  3. Hard refresh browser
  4. Try a different browser

Problem: Links don't work
────────────────────────
Solution:
  1. Check internet connection for blog links
  2. Verify blog URL is correct
  3. Clear browser cache

Problem: Animations are jerky
──────────────────────────────
Solution:
  1. Close other tabs/applications
  2. Update your graphics drivers
  3. Try a modern browser (Chrome, Edge, Firefox)
  4. Disable browser extensions

Problem: Page loads slowly
──────────────────────────
Solution:
  1. Clear browser cache
  2. Close other tabs
  3. Check internet speed
  4. Try a different browser
