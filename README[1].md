1. **index.html** - Main HTML structure (14 KB)

1. **Download all three files** to the same folder

## 🔧 Customization

### Change Blog Link
To change the blog URL, search for `threatscybersecurityinsider.blogspot.com` in all three files and replace with your desired URL.

### Change Contact Information
Edit the contact section in `index.html`:
```html
<div class="info-item">
    <span class="info-icon">📧</span>
    <div class="info-content">
        <span class="info-label">Email</span>
        <p>your-email@example.com</p>
    </div>
</div>
```

### Change Company Name
Search for "The Frostcod Labs" in the HTML and CSS to update branding.

### Change Colors
Edit CSS variables at the top of `styles.css`:
```css
:root {
    --primary-color: #00d4ff;
    --primary-dark: #0099cc;
    /* ... etc ... */
}
```

## 🐛 Troubleshooting

### CSS not loading
- Ensure `styles.css` is in the same folder as `index.html`
- Check file names match exactly (case-sensitive on Linux/Mac)
- Clear browser cache (Ctrl+Shift+Delete)

### JavaScript not working
- Ensure `script.js` is in the same folder as `index.html`
- Check browser console for errors (F12)
- Ensure JavaScript is enabled in browser

### Animations not smooth
- This is normal on very old browsers
- Try a modern browser (Chrome, Firefox, Edge)
- Check GPU acceleration is enabled

### Links not working
- Ensure you have an active internet connection (for blog links)
- Check blog URL is correct
- Links should open in new tab (target="_blank")
